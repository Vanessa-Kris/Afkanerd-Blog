SWOB OpenAPI

What is OpenAPI?

OpenAPI is an open source SMS queue management system.

It is used to manage the sending out of SMS messages to ensure none is lost along the way.


Is it free?

Yes! it's source code is open source.


Can I set it up myself?

The setup is developer-heavy, but there is documentation to help out.
